# 🚀 AWS-Deployed Flask Job Board

A lightweight Flask-based job board application deployed on an Amazon EC2 instance. This project demonstrates backend development, cloud deployment, and version control using Git and GitHub.

---

## 🌐 Live Demonstration
Previously deployed on AWS EC2. Available upon request.

---

## ✨ Features
- Built with Python and Flask
- Displays job listings via a web interface
- Deployed on an AWS EC2 Ubuntu server
- Configured networking, SSH access, and security groups
- Managed dependencies using virtual environments
- Version-controlled using Git and GitHub

## ▶️ How to Run Locally

### 1. Clone the Repository
```bash
git clone https://github.com/aebduelrehman-dev/flask-job-board-aws.git
cd flask-job-board-aws
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Application
```bash
python app.py
```

### 4. Open in Your Browser
```
http://127.0.0.1:5000
```

## ☁️ AWS Deployment Overview
- Launched and configured an AWS EC2 Ubuntu instance
- Enabled secure SSH access using a key pair

- Deployed and ran the Flask application on the cloud
- Managed dependencies using a Python virtual environment
